# 钉钉 OpenAPI 接入前置准备

> 在跑自动化脚本之前，您需要先在钉钉开放平台准备 4 项凭证。
> 这一步只做一次，之后脚本可重复使用。

---

## 1. 角色分工

| 角色 | 谁 | 做什么 |
|---|---|---|
| **企业管理员** | 您的钉钉组织超管 | 在 [钉钉管理后台](https://oa.dingtalk.com) 开通「智能表格」应用 + 分配权限 |
| **开发者**（您） | 自己 | 在 [钉钉开放平台](https://open-dev.dingtalk.com) 注册开发者账号、创建应用、拿凭证 |

---

## 2. 步骤 1：注册开发者账号（5 分钟）

1. 打开 https://open-dev.dingtalk.com
2. 用您的**钉钉账号**扫码登录
3. 进入「开发者后台」
4. 如果提示「完善开发者信息」，按提示填：手机号、邮箱、所在企业

---

## 3. 步骤 2：创建企业内部应用（10 分钟）

1. 开发者后台 → 顶部「应用开发」→「企业内部应用」→「创建应用」
2. 填写：
   - 应用名称：`面料推荐自动化`
   - 应用图标：随便选一个
   - 应用描述：`通过 OpenAPI 自动创建并维护面料推荐 AI 表格`
   - 应用类型：选 **H5 微应用** 或 **机器人**（任选，H5 更通用）
3. 创建完成后，**记下以下 3 个值**（待会要填到 `.env`）：
   - **AppKey**（在「凭证与基础信息」页面）
   - **AppSecret**（点击「查看」获取，只显示一次，要保存好）
   - **AgentId**（在「基础信息」页面，应用卡片右下角）

---

## 4. 步骤 3：申请 AI 表格权限（关键！10-30 分钟）

这是最容易卡住的一步，权限不到位 API 会返回 403。

### 必需权限
| 权限名 | 权限 code | 说明 |
|---|---|---|
| 智能表格 - 读写 | `AITable.Write` 或类似 | 创建/编辑表格 |
| 智能表格 - 读 | `AITable.Read` | 读取记录 |

### 申请方式
1. 在应用详情页 → 左侧「**权限管理**」
2. 搜索「**智能表格**」或「**AI 表格**」
3. 勾选所有相关权限
4. 点击「**申请权限**」

> ⚠️ **企业管理员审批**：
> - 如果您是普通员工，提交后需要等管理员审批（企业微信/钉钉通知）
> - 如果您就是管理员，会自动通过
> - 审批时效：从几分钟到 1-2 天不等

### 查看是否申请成功
- 应用详情 → 权限管理 → 列表中「智能表格」状态显示「**已开通**」

---

## 5. 步骤 4：获取 corpId（2 分钟）

corpId 是您的**企业 ID**，每个钉钉组织唯一。

**获取方式 3 选 1**：

### 方式 A：从管理后台
1. 登录 https://oa.dingtalk.com
2. 「我的企业」→「企业信息」
3. 找到「**CorpID（企业 ID）**」，形如 `ding5xxxxxx`

### 方式 B：从开发者后台
1. 开发者后台 → 顶部「我的组织」
2. 复制「**CorpId**」

### 方式 C：随便进一个企业内部应用
- URL 里 `ding5xxxxxx` 那段就是 corpId

---

## 6. 步骤 5：把 4 个凭证填到 .env（1 分钟）

```bash
cd C:\Users\zhaolei\Desktop\学习\面料推荐\钉钉AI表格\openapi
copy .env.example .env
notepad .env
```

填入：
```ini
DINGTALK_CORP_ID=ding5xxxxxxxxxxxx
DINGTALK_APP_KEY=your_app_key_here
DINGTALK_APP_SECRET=your_app_secret_here
DINGTALK_AGENT_ID=1234567890
```

**保存后不要把 .env 上传到任何代码仓库。**

---

## 7. 验证前置是否就绪（2 分钟）

跑这个命令：
```bash
python test_auth.py
```

**期望输出**：
```
[DINGTALK] get_access_token ...
[200] access_token received, expires in 7200s
✅ 鉴权 OK
```

**如果报错**：
| 错误 | 原因 | 解决 |
|---|---|---|
| `invalid appKey or appSecret` | 凭证填错 | 重新核对 .env |
| `permission denied` | 权限没开通 | 回到步骤 3 重新申请 |
| `corpId not found` | corpId 填错 | 重新获取 |
