# 面料推荐 · 钉钉 AI 表格 · 搭建手册

> 目标：在钉钉工作台创建「面料推荐总表」AI 表格，导入 86 条数据，配置 4 项 AI 能力，搭 4 个视图。
> 适用版本：钉钉 7.x 及以上（需开启「智能伙伴」/「AI 表格」权限）
> 预计耗时：45-60 分钟（首次配置）

---

## 0. 前置准备

| 物料 | 路径 | 说明 |
|---|---|---|
| 空白模板 | `面料推荐_钉钉AI表格_导入模板.xlsx` | 字段说明 + 字典 |
| 预转数据 | `面料推荐_预转数据_86条.xlsx` | 86 条，28 列 |
| 备选 CSV | `面料推荐_预转数据_86条.csv` | 编码 UTF-8-BOM |
| 表结构设计 | `01_表结构设计.md` | 字段说明、字典、视图设计 |

**权限要求**：
- 你需要是钉钉组织管理员，或有「智能表格」应用创建权限
- 需要开通「钉钉智能伙伴」（部分 AI 字段需要）

---

## 1. 创建 AI 表格（5 分钟）

### 步骤 1.1：进入工作台
1. 打开钉钉 PC 端 → 左侧「工作台」
2. 顶部搜索框输入「**AI 表格**」或「**智能表格**」
3. 点击进入应用（如果没有，请联系管理员开通）

### 步骤 1.2：新建表格
1. 点击右上角「**+ 新建**」→「**AI 表格**」
2. 命名：**面料推荐总表**
3. 描述：*工厂/打样师参考用，含 5 家供应商 86+ 款面料，支持智能搜索、卖点生成、识图录入*
4. 权限：
   - 组织内可见 ✅
   - 编辑权限：管理员 + 采购组 ✅
   - 查看权限：工厂 / 打样组 / 业务组（只读）✅
5. 点击「**创建**」

---

## 2. 配置 25 个字段（15 分钟）

进入表格后，默认显示一个空表。**不要用默认字段**，按下面的步骤重建：

### 步骤 2.1：删除默认字段
右键表头 → 「删除列」× 默认字段

### 步骤 2.2：按顺序创建字段
**最快的方法：用 Excel 导入表头**

1. 表格右上角「···」→「**从 Excel 创建**」
2. 选择 `面料推荐_钉钉AI表格_导入模板.xlsx` 的「**模板_导入用**」Sheet
3. 钉钉会自动识别 28 列并创建，类型映射如下：

| Excel 列名 | 钉钉字段类型 |
|---|---|
| 面料编号、面料名称、成分描述、其他成分、阻燃等级、备注、__来源文件、__导入时间 | 文本 / 日期 |
| 面料图片 | 附件 |
| 涤纶 %、棉 %、氨纶 %、再生纤维 %、幅宽 cm、克重 g/㎡、RMB 价格、FOB 价格、起订量 MOQ、__来源行号 | 数字 |
| 品类、供应商、结构/织法 | 单选（钉钉会自动从示例值识别为单选） |
| 适用季节、推荐款式、特性标签、后整理 | 多选（用分号分隔的，会被自动识别为多选标签） |
| 卖点文案 | 长文本 |
| 相似款链接 | 关联记录（本表自关联） |

4. 核对每列类型，**手动修正单选/多选字段**（钉钉自动识别不一定准）：
   - 「品类」→ 改为**单选** → 字典见 `字典` Sheet
   - 「供应商」→ 改为**单选**
   - 「结构/织法」→ 改为**单选**
   - 「适用季节」→ 改为**多选标签**
   - 「推荐款式」→ 改为**多选标签**
   - 「特性标签」→ 改为**多选标签**
   - 「后整理」→ 改为**多选标签**

> **提示**：在「多选标签」字段设置中，把 `字典` Sheet 里对应的值粘贴进去，作为可选值。

### 步骤 2.3：设置单选字典
- 「品类」：针织、机织、PU 绒、家纺（4 个）
- 「供应商」：常熟市华瑞针纺织、中涛 / 三时、万泰、home_fr、3S AVVA（5 个）
- 「结构/织法」：纬编针织、经编针织、平纹、斜纹、缎纹、提花、灯芯绒、摇粒绒复合、PU 涂层（9 个）

### 步骤 2.4：设置面料编号为唯一
- 「面料编号」→ 字段设置 → 勾选「**不允许重复**」
- 这样导入重复的会被钉钉自动拒绝

## 3. 导入 86 条数据（5 分钟）

### 步骤 3.1：删除样例行
打开 `面料推荐_钉钉AI表格_导入模板.xlsx`，**删除第 2 行**（样例行）

### 步骤 3.2：直接用预转数据导入
1. 表格右上角「···」→「**导入 Excel/CSV**」
2. 选择 `面料推荐_预转数据_86条.xlsx`（86 条）
3. 字段映射确认 → 全部「**匹配**」→ 下一步
4. 重复处理策略：
   - 「面料编号」**跳过**（避免重复）
   - 其他字段**覆盖**
5. 点击「**开始导入**」
6. 等待 1-2 分钟，导入完成后会显示「导入成功 86 条」

### 步骤 3.3：验证
- 检查总行数应为 86
- 抽查 3-5 条，确认成分、克重、供应商、季节、款式都正确

## 4. 配置 4 项 AI 能力（20 分钟）

### AI 能力 1：智能补全字段 ⭐

**作用**：在「面料名称」输入新品名，AI 自动补全成分、克重、幅宽等

**配置步骤**（以「成分描述」列为例）：

1. 选中「成分描述」列 → 字段设置 → 「**AI 字段**」
2. 类型选「**智能补全**」
3. 触发条件：「**当 [面料名称] 改变时**」
4. Prompt 模板（**复制粘贴**）：
   ```
   根据面料名称「{面料名称}」和品类「{品类}」，
   推断该面料的典型成分配比（百分比）。
   返回格式示例：94%涤纶 + 6%氨纶
   如果无法推断，返回空字符串。
   ```
5. 输出格式：文本
6. 同样给「幅宽 cm」「克重 g/㎡」「结构/织法」「后整理」配置智能补全。

### AI 能力 2：按需求语义搜索 ⭐⭐（核心）

**作用**：在表格顶部搜索框输入"夏季 T 恤、280g 以下" → 自动筛出匹配面料

**配置步骤**：

1. 表格顶部「+」→「**新建视图**」→「**AI 智能视图**」
2. 命名：**「AI 智能筛选」**
3. 启用「**自然语言筛选**」
4. 语义映射配置（钉钉后台）：

   | 自然语言关键词 | 转换成的筛选条件 |
   |---|---|
   | 夏季 / 春秋 / 秋冬 | 适用季节 包含对应季节 |
   | T 恤 / 外套 / 卫衣 / 裤子 | 推荐款式 包含对应款式 |
   | 280g 以下 / 轻 / 厚 | 克重 数值比较 |
   | 莫代尔 / 棉 / 涤 / 氨纶 | 成分描述 包含 |
   | 莫代尔 / 棉 / 涤 / 氨纶 | 涤纶/棉/氨纶% 数值过滤 |
   | 摇粒绒 / 灯芯绒 | 面料名称 包含 |
   | 防水 / 阻燃 / 抗皱 | 特性标签 包含 |
   | 针织 / 机织 | 品类 ＝ |
   | 便宜 / 高端 | RMB 价格 排序 |

5. 启用「**语义联想**」：用户输入「春装」→ 自动联想到 春 + 外套 + 卫衣

### AI 能力 3：自动生成卖点文案 ⭐⭐⭐（最常用）

**作用**：每行卖点文案自动生成 30-80 字工厂/打样师视角推荐话术

**配置步骤**：

1. 选中「卖点文案」列 → 字段设置 → 「**AI 字段**」
2. 类型选「**文本生成**」
3. 触发条件：「**行创建时** + 「**手动点击 AI 按钮**」（双触发，避免每次修改都重算）
4. Prompt 模板（**复制粘贴**）：
   ```
   你是一位资深面料销售，请根据以下面料信息写一段
   30-80 字的工厂/打样师视角推荐话术。

   要求：
   1) 突出适用场景和工艺要点
   2) 提到至少 1 个替代关系或升级点
   3) 语气专业、简洁
   4) 不超过 80 字
   5) 用中文，分 1-2 段

   面料信息：
   - 名称：{面料名称}
   - 品类：{品类}
   - 成分：{成分描述}
   - 克重：{克重 g/㎡} g/㎡
   - 幅宽：{幅宽 cm} cm
   - 适用季节：{适用季节}
   - 推荐款式：{推荐款式}
   - 特性：{特性标签}
   ```
5. 输出格式：长文本
6. 启用「**批量生成**」：导入数据时一次性为所有行生成

### AI 能力 4：图片识别录入 ⭐

**作用**：上传面料图片 → AI 自动识别成分、克重、织法

**配置步骤**：

1. 选中「面料图片」列 → 字段设置 → 「**AI 字段**」
2. 类型选「**图片理解**」
3. 启用「**识别后自动填入**」→ 勾选「成分描述」「克重 g/㎡」「结构/织法」
4. Prompt 模板（**复制粘贴**）：
   ```
   请识别这张面料图片，输出以下信息（用 JSON 格式）：
   {
     "成分描述": "推测成分，如 94%涤+6%氨",
     "克重": 380,
     "结构": "如平纹/斜纹/纬编/经编"
   }

   如果图片太模糊无法判断，对应字段返回 null。
   ```
5. 保存

### AI 能力 5（附加）：找相似款

**作用**：在「相似款链接」列点击 AI 按钮 → 找出 3-5 款同品类/克重接近的替代款

**配置步骤**：

1. 选中「相似款链接」列 → 字段设置 → 「**AI 字段**」
2. 类型选「**关联推荐**」
3. 关联目标：当前表
4. 匹配规则：
   - 相同「品类」✅
   - 克重差 ±20% ✅
   - 至少 1 个共同「推荐款式」✅
   - 排除自身 ✅
5. 排序：克重差绝对值 升序
6. 最多返回：5 条
7. 保存

## 5. 配置 4 个视图（10 分钟）

### 视图 1：总表（默认）
- 显示所有 25 列
- 排序：供应商 asc，品类 asc
- 分组：供应商

### 视图 2：按季节推荐 ⭐（工厂用得最多）
1. 新建视图 → 命名「**按季节推荐**」
2. 隐藏列：__来源文件、__来源行号、__导入时间、面料编号、阻燃等级、起订量 MOQ
3. 显示列：面料图片、面料名称、品类、成分、克重、幅宽、适用季节、推荐款式、卖点文案、RMB 价格
4. 默认筛选：适用季节 包含「当前月份」对应的季节
   - 3-5 月 → 春
   - 6-8 月 → 夏
   - 9-11 月 → 秋
   - 12-2 月 → 冬
5. 排序：克重 g/㎡ asc

### 视图 3：阻燃 / 特殊功能专版
1. 筛选：阻燃等级 不为空 **或** 特性标签 包含「阻燃」
2. 突出显示：阻燃等级、成分描述、克重

### 视图 4：新品入库视图（采购用）
1. 筛选：__导入时间 大于「30 天前」
2. 排序：__导入时间 desc
3. 显示列：面料编号、面料名称、供应商、成分、克重、起订量、RMB 价格、__导入时间

| 角色 | 权限 |
|---|---|
| 管理员（你 + 采购主管） | 完全权限：增删改、配置 AI 字段、导入数据 |

- 按角色配置，参考上表

## 10. 常见问题

**Q1：AI 字段不生效？**
A：检查钉钉版本是否 ≥ 7.0，并确认「智能伙伴」已开通。联系管理员。

**Q2：导入数据后排版乱了？**
A：钉钉的列宽有时会被自动调整。手动拖一下，或重新导入。

**Q3：卖点文案生成不专业？**
A：调整 Prompt 模板，加入更多业务关键词（如"工厂""打样""克重""替代"）。

**Q4：图片识别准确率低？**
A：拍图时尽量用纯色背景、避免褶皱。可以在 Prompt 里加「如果无法判断，返回 null」。

**Q5：搜索结果太多？**
A：组合筛选：在 AI 智能视图里加二次筛选条件（如克重范围、品类）。
