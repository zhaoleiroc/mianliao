# 面料推荐 · 钉钉 AI 表格 · 表结构设计

> 面向：工厂 / 打样师参考
> 数据源：现有 Excel/JSON 面料库（86 条，5 家供应商）
> 钉钉版本：钉钉 AI 表格（DingTalk AITable）— 6.x 及以上
> 设计日期：2026-06-30

---

## 一、设计目标

把现有分散的 5 家供应商 Excel 汇总成一张「面料推荐总表」，让工厂/打样师在钉钉内可以：
1. **一站式查询**所有面料的成分、克重、幅宽、价格
2. **按需求语义搜索**（如"夏季女装 T 恤、280g 以下、莫代尔混纺"）
3. **看 AI 自动生成的卖点文案**（直接转发给客户）
4. **看图识料**，现场拍面料图片直接录入
5. **找相似款**（替代款 / 升级款）

## 四、AI 能力映射

### 1. 智能填写 / 补全字段
**触发方式**：在「面料名称」列输入新品名 → AI 自动补全成分、克重、幅宽等

**钉钉配置**：
```
列：成分描述 / 幅宽 / 克重 / 结构 / 后整理
类型：AI 字段 → 「智能补全」
prompt 模板：
  "根据面料名称【{面料名称}】和品类【{品类}】，
   推断该面料的典型成分、克重范围、幅宽、织法和后整理。
   如果是新品无信息，返回 null，不要编造。"
```

### 2. 按需求语义搜索 / 筛选
**触发方式**：表格顶部搜索框输入自然语言 → AI 转成筛选条件

**钉钉配置**：
- 表格视图 → 添加「AI 智能筛选视图」
- 绑定自然语言解析，匹配以下语义：
  - 季节：`夏季` → 适用季节 包含"夏"
  - 款式：`女装 T 恤` → 推荐款式 包含"T 恤" + 特性 包含"亲肤"
  - 成分：`莫代尔` → 成分描述 包含"莫代尔"
  - 克重：`280g 以下` → 克重 ≤ 280

### 3. 自动生成推荐理由 / 卖点文案
**触发方式**：新建一行或在「卖点文案」列点击 AI 按钮 → 自动生成 30-80 字推荐话术

**钉钉配置**：
```
列：卖点文案
类型：AI 字段 → 「文本生成」
prompt 模板：
  "你是一位资深面料销售，请根据以下信息写 30-80 字的
   工厂/打样师视角推荐话术，重点突出适用场景、工艺要点、
   替代关系。语气专业、简洁。

   面料名称：{面料名称}
   成分：{成分描述}
   克重：{克重} g/㎡
   幅宽：{幅宽} cm
   适用季节：{适用季节}
   推荐款式：{推荐款式}
   特性：{特性标签}

   输出格式：纯文本，1-2 段，不超过 80 字。"
```

### 4. 图片识别录入
**触发方式**：上传面料图片 → AI 自动识别并填入成分、克重、织法、可能品类

**钉钉配置**：
```
列：面料图片
类型：AI 字段 → 「图片理解」
绑定后续字段：成分描述 / 克重 / 幅宽 / 结构
prompt 模板：
  "请识别这张面料图片，输出：
   1) 推测成分（如棉/涤/混纺）
   2) 推测克重范围（g/㎡）
   3) 推测织法（平纹/斜纹/针织/提花）
   4) 推测幅宽范围（cm）
   如果图片太模糊无法判断，请明确写出'无法识别'。"
```

### 5. 找相似款（附加价值）
**触发方式**：在「相似款链接」列点击 AI 按钮 → 找出 3-5 款同品类 / 同克重 / 同应用场景的替代款

**钉钉配置**：
```
列：相似款链接
类型：AI 字段 → 「关联推荐」
匹配规则：
  - 相同品类
  - 克重 ±20%
  - 至少 1 个共同「推荐款式」
  - 排除自身
排序：克重接近度 desc，最多 5 条
```

## 五、视图设计（4 个视图）

### 视图 1：总表（默认）
所有面料 25 列全展示，按「供应商 + 品类」分组。

### 视图 2：按季节推荐 ⭐（工厂用得最多）
显示列：图片、面料名称、成分、克重、幅宽、价格、卖点文案
筛选：适用季节 ＝ 当前月份对应季节（自动随月份切换）

### 视图 3：阻燃 / 特殊功能专版
显示列：图片、面料名称、成分、阻燃等级、价格
筛选：阻燃等级 ≠ 空 或 特性标签 包含"阻燃"

### 视图 4：新品入库视图（采购用）
显示列：编号、名称、供应商、成分、克重、起订量、录入时间
排序：导入时间 desc，仅显示最近 30 天新增
